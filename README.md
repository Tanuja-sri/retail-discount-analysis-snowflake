# 🛒 Retail Discount Analysis using Snowflake

## 📌 Problem Statement

Calculate order amount and apply discount based on purchase value.

## 🧾 Tables Used

* products
* customers
* orders

## ⚙️ Logic

* Amount = product_price × quantity
* Apply 10% discount if amount ≥ 50,000
